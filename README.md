### Requirements

+ python 3.5

+ pyqt5 

+ pytube 

+ ffmpeg


### How to run



1. download all files.


2. run main.py


3. if you want to convert : macOS -> install ffmpeg using home brew ; Windows -> download ffmpeg


### Precautions


+ If you download too much, you are likely to be banned from YouTube.
